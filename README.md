# 3D 交互式粒子系统

基于 Three.js 和 MediaPipe 的实时手势控制 3D 粒子系统

## 功能特性

- ✋ **手势控制**: 通过摄像头检测双手张合控制粒子群的缩放和扩散
- 🎨 **多种模型**: 支持圣诞树/星星/烟花/爱心/地球/球体等粒子模型
- 🌈 **颜色自定义**: 使用颜色选择器自由调整粒子颜色
- ⚡ **实时响应**: 粒子实时响应手势变化
- 🖥️ **全屏模式**: 支持全屏显示
- 🎯 **简洁界面**: 现代化 UI 控制面板

## 技术栈

- **Three.js**: 3D 渲染引擎
- **MediaPipe**: 手部关键点检测
- **Vite**: 快速开发构建工具

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

## 使用说明

1. **摄像头权限**: 首次打开页面请允许浏览器访问摄像头
2. **手势控制**:
   - 使用双手张开或收合来控制粒子系统缩放
   - 单手张开程度也可以控制粒子大小
3. **模型切换**: 通过顶部控制面板选择不同粒子模型
4. **颜色调整**: 使用颜色选择器自定义粒子颜色
5. **粒子数量**: 通过滑块调整粒子数量（100-5000）
6. **全屏模式**: 点击右下角全屏按钮进入全屏显示
